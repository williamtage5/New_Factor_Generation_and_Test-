# 结果分析报告

## 1. 报告说明

- 样本区间：`2015-01-05` 至 `2023-12-29`。
- 样本范围：沪深300成分股日频截面，每个交易日固定 `300` 只股票。
- 回测口径：单因子 IC 分析 + 十分位分组回测。
- IC 滞后期：`1/5/10/20` 个交易日。
- 分组方式：按因子值从低到高分为 `10` 组，重点观察 `G1`、`G10` 与 `LS_Gn_G1`。
- 数据说明：以下因子名中的 `clip3` 均表示经过横截面 `3 sigma` 截尾处理。
